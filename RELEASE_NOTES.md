# Skill Grove v0.5.1

Released: 2026-05-08

## Highlights

- Added skill sharing for real sources: create a soft link for the current skill in another configured Agent.
- Added ZIP export for sharing a real skill source as a portable package.
- Added content governance for same-content skills: convert duplicate real directories into soft-link entries.
- Real directories are moved to Trash before conversion, so the operation remains locally recoverable.

## macOS

- This release publishes a macOS DMG installer only.
- Download the `.dmg` asset below and drag Skill Grove into Applications.
- Users on v0.5.0 can use the in-app updater to install this release.
