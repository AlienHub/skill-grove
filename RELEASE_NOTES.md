# Skill Studio v0.2.0

Released: 2026-05-07

## Highlights

- Reworked the skill detail page around content versions and Agent sources, making mismatched source content much easier to compare.
- Added a grouped current-source picker with search by Agent or path.
- Added current-source actions for opening directories in Finder, IDEs, or editors.
- Added source cleanup actions: soft links can be deleted, and real source directories are moved to the macOS Trash.
- Fixed browser development support for source cleanup and dropdown layering issues around source menus.

## macOS

- This release publishes a macOS DMG installer only.
- Download the `.dmg` asset below and drag Skill Studio into Applications.
- Users on v0.1.3 can use the in-app updater to install this release.
