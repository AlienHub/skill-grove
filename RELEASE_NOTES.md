# Skill Grove v0.6.4

Released: 2026-06-16

## Highlights

- **Daily skill call stats:** Application View now shows per-day skill call activity for each Agent app, including a stacked bar breakdown of which skills were called each day.
- **Scan toggles:** Built-in and custom Agent skill directories can be paused from scanning without removing them as share targets.
- **Multi-target sharing:** Sharing a skill to other Agent directories now supports selecting multiple targets at once, including Claude Code.
- **Smoother config UI:** Scan switch updates no longer flash the whole configuration screen; only the clicked switch enters a pending state.

## macOS

- This release publishes a macOS DMG installer only.
- Download the `.dmg` asset below and drag Skill Grove into Applications.
- Users on v0.6.3 can use the in-app updater to install this release.
